# Vibe Coding x JO Paris 2024

Vibe coded web site for ynov e-learnings, built with [Next.js](https://nextjs.org) and deployed on [Vercel](https://vercel.com).

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run start` | Start production server |
| `npm run lint` | Run ESLint |

## Deploy on Vercel

Deploy this project with one click:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

Or push to your GitHub repository — Vercel will automatically detect Next.js and deploy it.
